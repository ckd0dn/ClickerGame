# ClickerGame
유니티 게임개발 심화주차 개인과제 (클리커 게임)

## 🖥️ 프로젝트 소개
마우스로 클릭할때마다 몬스터를 공격하고 Gold를 얻어 공격력을 강화하고 
스테이지별로 강화되는 몬스터를 무한정으로 잡는 게임

## 🕶 기본 조작법
마우스 왼쪽 클릭: 공격
